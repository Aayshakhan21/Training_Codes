# JSP_Programs

## Concept and Topic Notes
This project demonstrates JSP fundamentals and page-level concepts.
- JSP lifecycle basics and page rendering
- Request/response data display in JSP
- Separation of view logic from Java classes

## Key Annotations / Tags Used
- Project mainly uses XML/JSP configuration (no major Java annotations detected).

## Package / Module Overview
- `src/main/java` : Java source and module logic
- `src/main/webapp` : JSP/UI and web deployment descriptors

## How to Use
1. Import project in IDE.
2. Configure Tomcat and run web module.

## Objective
- Understand project-specific concepts, topics, and annotation usage through hands-on practice.
