# JDBCSpringMVCProject

## Concept and Topic Notes
This project demonstrates Spring MVC + JDBC architecture.
- Controller to Service/DAO request flow
- Form handling and view rendering with JSP
- Database integration through JDBC layer

## Key Annotations / Tags Used
- `@Autowired`
- `@Controller`
- `@GetMapping`
- `@ModelAttribute`
- `@Override`
- `@PathVariable`
- `@PostMapping`
- `@Repository`

## Package / Module Overview
- `src/main/java` : Java source and module logic
- `src/main/resources` : properties/xml configuration
- `src/main/webapp` : JSP/UI and web deployment descriptors

## How to Use
1. Import as Maven project.
2. Build with `mvn clean install`.
3. Deploy on Tomcat and run in browser.

## Objective
- Understand project-specific concepts, topics, and annotation usage through hands-on practice.
