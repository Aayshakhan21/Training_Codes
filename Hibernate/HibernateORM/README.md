# HibernateORM

## Concept and Topic Notes
This project demonstrates core Hibernate ORM concepts.
- Mapping Java classes to database tables
- Managing SessionFactory, Session, and Transaction
- Persisting and retrieving entities using Hibernate APIs

## Key Annotations / Tags Used
- `@Entity`
- `@GeneratedValue`
- `@Id`
- `@Override`
- `@SuppressWarnings`
- `@Table`

## Package / Module Overview
- `src/main/java` : Java source and module logic
- `src/main/resources` : properties/xml configuration

## How to Use
1. Import as Maven project.
2. Build with `mvn clean install`.
3. Run main/test classes from IDE.

## Objective
- Understand project-specific concepts, topics, and annotation usage through hands-on practice.
